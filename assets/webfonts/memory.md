# MEMORY

## Role

Polices web et fontes d'icones necessaires au rendu front.

## Contenu

- variantes de police du theme,
- ressources Font Awesome (`brands`, `regular`, `solid`) sous plusieurs formats.

## Vigilance

- Eviter les ajouts inutiles de formats dupliques.
- Si une police n'est plus utilisee, verifier toutes les references CSS avant suppression.
