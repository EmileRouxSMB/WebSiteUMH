# MEMORY

## Role

Feuilles de style consommees directement par les pages HTML.

## Contenu

- `main.css`: style principal du theme.
- `annuaire.css`: styles de l'annuaire partenaires.
- `form-prestataire.css`: styles du formulaire prestataire.
- fichiers de support comme `noscript.css` et Font Awesome.

## Vigilance

- Si un CSS est genere depuis SASS, conserver la correspondance de source.
- Documenter toute nouvelle feuille specialisee pour eviter les recouvrements.
