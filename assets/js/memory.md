# MEMORY

## Role

Scripts front du site.

## Contenu

- scripts metier (`annuaire.js`, `form-prestataire.js`, `promo-text.js`, `umh-config.js`).

## Vigilance

- Prioriser des scripts courts, commentes sur l'intention et separes par responsabilite.
