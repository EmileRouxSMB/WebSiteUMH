# MEMORY

## Role

Ressources front partagees du site.

## Contenu

- `css/`: feuilles compilees ou maintenues directement.
- `js/`: scripts front.
- `sass/`: sources de styles.
- `webfonts/`: polices et Font Awesome.

## Vigilance

- Eviter de melanger sources SASS et sorties CSS sans le documenter.
- Toute reorganisation des assets doit etre decrite ici.
