# MEMORY

## Role

Site web principal `Un Mariage Heureux`.

## Contenu

- Pages HTML publiques du site.
- `assets/`: styles, scripts et polices.
- `images/`: visuels du site et partenaires.
- `data/`: donnees locales de fallback.
- `docs/`: scripts ou documentation d'integration.
- `file/`: PDF telechargeables.

## Vigilance

- Toute evolution de structure, de flux de donnees ou d'integration externe doit etre reportee ici.
- Garder les pages racine concentrees sur le rendu et deleguer la logique aux fichiers `assets/`.
